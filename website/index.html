Garbanzo is a chat bot that lives in your group chats. It answers questions with live transit and weather data, welcomes new members, and keeps notes on what your community cares about. You run it on your own server, with the AI providers you choose.

Set up in a few minutes

Runs on any Docker host, including a Raspberry Pi. API keys and chat data stay on your machine.

# 1. configure
git clone https://github.com/jjhickman/garbanzo-bot.git
cd garbanzo-bot && npm run setup
# 2. run
docker compose up -d

What it does all day

Answers with live data

Ask about transit, weather, venues, news, or books in plain language. The bot looks the answer up and includes it in the reply, so there is no command syntax to learn.

Remembers your community

The bot keeps notes: who runs book club, where trivia night happens, which park hosts the picnics. You can add facts yourself or let it pick them up from conversation.

Keeps plans on track

When someone proposes a meetup, the bot adds weather and transit details, then reminds the chat before it starts. Owners also get a weekly summary on Sundays.

Moderation, human in charge

Flagged messages go to the owner for review instead of triggering automatic punishment. Edited messages get checked again, and repeat problems are tracked per member.

Routes across providers

Works with OpenAI, Anthropic, Google, AWS Bedrock, and OpenRouter. You pick the order; if one provider fails, the next takes over. Spending is tracked per provider, and you can serve simple replies from a local model through Ollama.

Shows you everything

An admin page shows daily spend and group activity. Prometheus metrics and a ready-made Grafana dashboard start alongside the bot with one extra compose flag.
