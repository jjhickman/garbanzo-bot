Features

Live lookups

Transit, weather, news, venues, books. Requested in any natural language!

Community lore

Add community facts yourself or collect them automatically. Everything is viewable and removable. Each community keeps its own lore; an instance shares selected lore with another community only when the owner permits it.

Events

Weather and transit attached to meetup plans. A reminder before start time. Sunday recap for owners.

Moderation

Flags go to the owner for review. Edited messages get re-checked. Strikes tracked per member. No automatic punishment.

AI providers

OpenAI, Anthropic, Google, AWS Bedrock, OpenRouter. Any order, automatic failover. Cost readout included. Ollama for local replies.

Monitoring

Prometheus and Grafana powered dashboards for observing community activity, token spend, and more.

Band mode

The same bot runs as Remy for bands on Discord. Song catalog, rehearsal reminders, setlists, and song ideas captured from voice memos. Off by default.

Backups

Nightly off-machine backups of credentials and data, verified after every run. One command restores.
