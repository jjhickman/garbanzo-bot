A group chat bot
you run yourself.

Garbanzo started as a bot for a Boston WhatsApp group. Somebody asks if the train is late, it checks and answers. It welcomes people who introduce themselves, keeps notes on what the group cares about, and reminds everyone before events. All of it runs on your own server, with whichever AI provider you give it a key for.

Running it

Your API keys and chat data stay on your machine, whichever way you host it:

  • Docker Compose: the usual way; runs on a Raspberry Pi, a home server, or a VPS
  • Kubernetes: the published image works on any cluster
  • AWS: CDK stacks for EC2 and ECS ship in the repo
  • Bare metal: a systemd unit or prebuilt binaries, no container at all
# 1. configure
git clone https://github.com/jjhickman/garbanzo-bot.git
cd garbanzo-bot && npm run setup
# 2. run
docker compose up -d

What it does

Live lookups

Somebody asks whether the train is delayed; the bot checks and answers right there. Weather, news, venues, and book lookups work the same way. No command syntax.

Group memory

It keeps notes on the group: who runs book club, which park hosts the picnics. Add facts yourself or let it collect them from conversation. You can see and delete everything it stores.

Events

When someone floats a meetup, the bot attaches the weather forecast and transit info, then reminds the chat before it starts. Owners get a summary of the week every Sunday.

Moderation

Flagged messages go to you, not to an auto-ban. Edited messages get re-checked. Repeat problems are tracked per member. The bot never punishes anyone on its own.

AI providers

Bring a key from OpenAI, Anthropic, Google, AWS Bedrock, or OpenRouter. Put them in whatever order you like; if one fails, the next answers. There is a cost readout so you know what you are spending, and Ollama works too if you want simple replies handled locally for free.

Dashboards

An admin page shows daily spend and group activity. If you want more history, one compose flag starts Prometheus and Grafana with a dashboard already set up.
