Running it

API keys and chat data stay on your machine:

  • Docker Compose: Raspberry Pi, home server, or VPS
  • Kubernetes: published image, any cluster
  • AWS: CDK stacks for EC2 and ECS in the repo
  • Bare metal: systemd unit or prebuilt binaries
# 1. configure
git clone https://github.com/jjhickman/garbanzo-bot.git
cd garbanzo-bot && npm run setup
# 2. run
docker compose up -d

What it does

Live lookups

Transit, weather, news, venues, books. Plain-language questions, answered in the reply. No command syntax.

Group memory

Who runs book club, where the picnics are. Add facts yourself or let it collect them. Everything viewable and deletable.

Events

Weather and transit attached to meetup plans. A reminder before start time. Sunday recap for owners.

Moderation

Flags go to the owner for review. Edited messages get re-checked. Strikes tracked per member. No automatic punishment.

AI providers

OpenAI, Anthropic, Google, AWS Bedrock, OpenRouter. Any order, automatic failover. Cost readout included. Ollama for local replies.

Dashboards

Admin page for spend and activity. Prometheus and Grafana behind one compose flag.
