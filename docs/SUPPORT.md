## Owner Commands for Support Messaging

From owner DM:

- `!support` -> preview support message
- `!support broadcast` -> broadcast to all enabled groups

Support message content uses:

- `GITHUB_SPONSORS_URL`
- `PATREON_URL`
- `KOFI_URL`
- `SUPPORT_CUSTOM_URL`
- `SUPPORT_MESSAGE`
